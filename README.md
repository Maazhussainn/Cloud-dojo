# Cloud Dojo

Hands-on AWS learning platform MVP.

## Goal
Teach beginners AWS by *doing*, using guided labs, real AWS resources, and gamification.

## Features (MVP)
- User login / authentication (simple)
- Lab progress tracking
- Step-by-step lab instructions
- Deploy resources using AWS console (no automation required yet)

## Tech Stack
- Frontend: React (Vite or CRA)
- Backend: Node.js (later)
- Hosting (later stage): Amplify / S3 + CloudFront
